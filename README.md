### 主题特点

- 基于著名国产UI框架 AmazeUI
- 响应式设计
- 支持社交图标

### 主题安装

将主题文件夹改名为 bottle ，放置到 typecho 的 \usr\themes\ 目录下，并后台开启主题

### 主题设置

- 设置站点 Logo 地址
- 设置头像图片地址
- 设置作者简介
- 设置社交链接
- 后台添加统计代码
